ID 컬럼 규칙

1
등록 시 — ID 열은 비워두기
캠페인ID 등 시스템 발급 ID는 등록 시 빈칸. 서버가 자동 발급합니다.
2
수정 시 — 🟡 노란 ID 필수
수정 시트의 노란 ID 컬럼을 반드시 입력. 나머지는 바꿀 것만 입력.
3
삭제 시 — ID만 입력
삭제 시트에서 삭제할 항목의 ID만 입력합니다.
